
# Portafolio de Automatización de Pruebas Web - Juan Pablo Muñoz


Este repositorio contiene los proyectos de automatización de pruebas web que desarrollé durante mi proceso de aprendizaje y práctica en QA Automation.

El objetivo de este portafolio es demostrar mis habilidades para diseñar, implementar y mantener pruebas automatizadas confiables, asegurando la calidad del software mediante buenas prácticas de testing.


## Tecnologías que utilizo

- **Lenguajes:** JavaScript, TypeScript 
- **Frameworks de Automatización:** Playwright
- **Herramientas:** Git, GitHub, Node.js 

## Proyectos

### Stage 1: Fundamentos de Web Automation Testing

Challenge 1:
Automatización de un flujo básico en la aplicación CURA Healthcare Service utilizando Playwright.

Challenge 2:
Localizar elementos en una página web. 

Mission:
Implementación de una suite de pruebas automatizadas para validar un flujo de negocio completo en la tienda online TestingYes.
